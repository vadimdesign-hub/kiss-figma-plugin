# Kiss — Figma Plugin

Набор утилит для помощи в работе: выравнивание секций, управление приоритетами, поиск похожих объектов и другие инструменты работы в Figma. Плагин поддерживает компактную плавающую панель, через которую можно запускать любую команду в один клик — без необходимости каждый раз открывать меню Plugins.

## Функционал

### ⚡ Выровнять все секции
Автоматически выравнивает все секции на странице по сетке — расставляет их в ряды с одинаковыми отступами. Ничего не нужно выделять, плагин берёт все секции сам.

### ➡️ Расширить вправо / Дублировать вправо
Работает по-разному в зависимости от того, что выделено.

Если выделена **секция** — увеличивает её ширину на 620 px вправо, все секции правее сдвигаются.

Если выделен **фрейм внутри секции** — дублирует его вправо: секция расширяется на ширину фрейма + 80 px, клон ставится сразу после оригинала, правые секции сдвигаются.

### ⬅️ Расширить влево / Дублировать влево
Если выделена **секция** — увеличивает её ширину на 620 px влево (контент внутри сдвигается вправо), правые секции сдвигаются.

Если выделен **фрейм внутри секции** — дублирует его влево: клон встаёт на место оригинала, оригинал и все фреймы правее сдвигаются вправо, правые секции тоже сдвигаются.

### 📐 Выровнять + тёмная
Выравнивает фреймы внутри секции с одинаковыми отступами. Если фреймы высокие (больше 960 px) — автоматически дублирует их во второй ряд снизу и переключает дубли в тёмную тему через переменные Figma (Appearance → Dark).

На кнопке в панели инструментов есть чекбокс с иконкой клавиатуры — если его включить, перед выравниванием под каждым экраном автоматически добавится компонент клавиатуры.

### 🗂️ Обернуть + выровнять
Выдели любые объекты — плагин обернёт их в новую секцию и выровняет в один ряд. Если выделена уже существующая секция — просто выровняет содержимое внутри неё. Работает с любыми типами объектов: фреймы, компоненты, группы и т.д.

### 🔄 Замена instance
Выдели несколько объектов, последним — эталонный экземпляр. Плагин заменит все остальные выделенные объекты на копии эталона, сохранив их позиции.

### ✂️ Нарезка
Универсальный инструмент подготовки ассетов. Работает в трёх режимах в зависимости от чекбоксов на кнопке:

- **Без чекбоксов (по умолчанию)** — нарезка 1px. Каждый выделенный объект оборачивается во фрейм с отступом 1px, все результаты складываются в секцию «Нарезка».
- **Чекбокс «2.67»** — масштабирование. Каждый выделенный объект увеличивается в 2.67 раз, размеры округляются до чётных чисел. Результаты помещаются в секцию «Нарезка / Сервер».
- **Чекбокс «Продукт»** — серверные иконки. Каждый объект оборачивается во фрейм с отступом 1px, создаются три масштабированные копии — 80×80, 120×120 и 240×240. Результаты складываются в секцию «Нарезка / Продукт».

Чекбоксы взаимоисключающие — можно выбрать только один. После выполнения команды чекбокс автоматически сбрасывается.

### 🖼️ Размеры в арт
Автоматически проставляет размеры артов. Первый текстовый слой показывает реальный размер (W×H px), второй — итоговый размер арта в нужном масштабе.

### 🔍 Найти похожие
Выдели любой объект — фрейм, компонент, прямоугольник, что угодно. Плагин найдёт по всей странице объекты с таким же именем и такими же размерами и выделит их все, включая исходный.

### 🟠 Средний приоритет / 🔴 Высокий приоритет / 🟢 Задача выполнена / 🔵 На ревью
Четыре тега статусов, которые добавляются к выделенному объекту. Если тег уже есть — обновляет цвет и текст. Образуют простую систему статусов прямо внутри Figma. На панели инструментов отображаются компактной сеткой 2×2.

### 🚀 Ready for Dev
Помечает секции официальным статусом Figma «Ready for Dev» — разработчики видят его в режиме Dev Mode.

- Если выделены секции — переключает статус только у них.
- Если ничего не выделено — переключает все секции на странице: все помечены → снимает; хотя бы одна не помечена → ставит всем.

### 🎛️ Панель инструментов
Открывает компактную плавающую панель со всеми командами плагина в виде кнопок. Запускаешь один раз — панель висит поверх канваса. По умолчанию свёрнута (5 кнопок), стрелка **›** раскрывает все команды.

Кнопки можно перетаскивать для изменения порядка — порядок сохраняется между сессиями.

### ⚙️ Настройки
Открывает окно настроек плагина:

- **Стиль иконок** — переключает иконки панели инструментов и FAQ между SVG (монохромные линейные) и Emoji (цветные).
- **Тема** — переключает оформление между светлой и тёмной темой. Тема применяется к панели инструментов, настройкам и FAQ.

Все настройки сохраняются и применяются автоматически.

### 📖 FAQ
Открывает встроенную справку по всем командам плагина прямо внутри Figma. Каждая команда сопровождается превью кнопки как она выглядит на панели. Тема и стиль иконок FAQ соответствуют выбранным в настройках.

## Особенности

- Не требует API-ключей и внешних сервисов — всё работает локально внутри Figma
- Работает с любыми типами объектов: фреймы, компоненты, главные компоненты, группы
- Поиск по всей странице, включая содержимое секций
- Переключение тёмной темы через переменные Figma (Variable Modes / Appearance)
- Компактная плавающая панель инструментов с перетаскиванием кнопок
- Два стиля иконок (SVG / Emoji) и две темы (светлая / тёмная)
- Встроенный FAQ с превью кнопок и описанием каждой команды

## Как установить

1. Скачай или склонируй репозиторий
2. Открой Figma → Plugins → Development → Import plugin from manifest
3. Выбери `manifest.json` из папки проекта
4. Плагин появится в списке плагинов разработки
